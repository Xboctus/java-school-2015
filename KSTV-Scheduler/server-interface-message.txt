 - Сообщение — (синтаксически) последовательность частей сообщения.
 - Часть сообщения — (лексически) последовательность символов, не содержащая ограничителей сообщения и ограничителей части сообщения.
 - Ограничитель сообщения — символом U+001E.
 - Ограничитель части сообщения — символом U+001F.
 - Формат отправки сообщения сервером по сокету и HTTP:
	(часть_сообщения + ограничитель_части_сообщения)* + ограничитель_сообщения.
 - Формат приёма сообщения сервером по сокету:
	(часть_сообщения + ограничитель_части_сообщения)* + остаток_сообщения + ограничитель_сообщения.
 - Формат приёма сообщения сервером по HTTP:
	(часть_сообщения + ограничитель_части_сообщения)* + остаток_сообщения + ограничитель_сообщения?.
 - Остаток сообщения лексически имеет форму части сообщения (не завершённой ограничителем части сообщения) и игнорируется сервером.
