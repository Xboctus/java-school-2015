= Тип данных, их описание и представление =

Boolean
	Оп-е: Логическое значение
	Пр-е: "true" или "false"

Unsigned
	Оп-е: Целое неотрицательное число
	Пр-е: Число в десятичной записи без ведущих нулей

Port
	Оп-е: TCP-порт
	Пр-е: Число в десятичной записи без ведущих нулей

Text
	Оп-е: Произвольная последовательность символов
	Пр-е: Как есть

UserName
	Оп-е: Имя пользователя
	Пр-е: Непустая последовательность букв [a-zA-Z_] и цифр [0-9], первый символ из которых — буква

LocalDateTime
	Оп-е: Локальные дата-время
	Пр-е: YYYY-MM-DD hh:mm:ss

AbsTimeZone
	Оп-е: Временная зона в абсолютной записи (н-р, GMT+3)
	Пр-е: См. http://docs.oracle.com/javase/7/docs/api/java/util/TimeZone.html
