= (Описание действия) =
Метод - HTTP-метод, используемый клиентом для совершения действия
Клиент - параметры, передаваемые клиентом серверу
Сервер - параметры, возвращаемые сервером клиенту
Коды состояния - варианты кода состояния HTTP, возвращаемого сервером клиенту
(Необязательные параметры обозначены как [par=<par>])

= Тестовое действие =
(Клиент посылает на сервер логин, сервер сообщает, есть ли этот логин в его БД, и через 10 секунд посылает клиенту этот же результат как сообщение о событии в формате result=<result>.)
Метод: POST
Клиент:
	action = test
	login = <login>
	listen_port = <port> - прослушиваемый клиентом порт
Сервер:
	result = <result>
	listen_port = <port> - прослушиваемый сервером порт
<result>:
	yes - логин существует
	no - логин не существует
Коды состояния:
	400 Bad Request - лексическая ошибка
	200 OK - логин существует
	451 - логин не существует

= (Для всех действий) =
Коды состояния:
	200 OK - запрос выполнен успешно
	400 Bad Request - лексическая ошибка (неизвестное действие, лексически некорректное имя пользователя или пароль, ошибка в названии временной зоны и т.п.)
	401 Unauthorized - неверная пара логин-пароль для начала сессии, не начатая сессия - для всех других действий кроме запроса информации

= Начало сессии =
Метод: PUT
Клиент:
	action = start_session
	login = <login> - логин пользователя
	password = <password> - пароль этого пользователя
	listen_port = <port> - прослушиваемый клиентом порт

= Окончание сессии =
Метод: PUT
Клиент:
	action = stop_session

= Добавление пользователя =
Метод: PUT
Клиент:
	action = create_user
	login = <login> - логин пользователя
	password = <password> - пароль этого пользователя
	timezone = <timezone>
	[active = <active>] - по умолчанию true
	listen_port = <port> - прослушиваемый клиентом порт
Коды состояния:
	450 - такой логин уже существует

= Изменение информации о пользователе =
Метод: PUT
Клиент:
	action = change_user
	[timezone = <timezone>]
	[active = <active>]
	[newpassword = <newpassword>]

= Запрос информации о пользователе =
Метод: GET
Клиент:
	action = user_info
	login_from = <login>
Сервер:
	timezone = <timezone>
	active = <active>
	datetime = <datetime>
	text = <text>
	datetime = <datetime>
	text = <text>
	...
Коды состояния:
	451 - запрашиваемый логин не существует

= Добавление события =
Метод: PUT
Клиент:
	action = add_event
	datetime = <datetime>
	text = <text>
Коды состояния:
	460 - событие с таким текстом у этого пользователя уже есть

= Удаление события =
Метод: DELETE
Клиент:
	action = remove_event
	text = <text>
Коды состояния:
	461 - события с таким текстом у этого пользователя нет

= Добавление случайного события =
Метод: POST
Клиент:
	action = add_random_event
	text = <text>
	datetime_from = <datetime>
	datetime_to = <datetime>
Коды состояния:
	460 - событие с таким текстом у этого пользователя уже есть

= Клонирование события =
Метод: PUT
Клиент:
	action = clone_event
	login_from = <login>
	text = <text>
Коды состояния:
	451 - запрашиваемый логин не существует
	461 - события с таким текстом у пользователя login_from нет
	460 - событие с таким текстом у этого пользователя уже есть
